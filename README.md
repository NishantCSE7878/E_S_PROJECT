6 Machine Learning Projects

A comprehensive collection of 6 Computer Vision and Machine Learning projects developed in Python. This suite covers everything from Attendance Systems and Drowsiness Detection to Traffic Analysis and Sign Language Recognition.

## 🚀 Projects Included

1.  **Attendance System:** Face recognition with emotion detection and CSV logging (9:30 AM - 10:00 AM constraint).
2.  **Animal Detection:** YOLOv8-based model that highlights carnivores in red and triggers pop-up alerts.
3.  **Drowsiness Detection:** Real-time eye-tracking and age prediction for vehicle safety.
4.  **Nationality Predictor:** Complex attribute detection (Race, Emotion, Age, and Dress Color) based on specific nationality logic.
5.  **Sign Language Recognition:** A trained CNN model to recognize hand gestures (Operational 6 PM - 10 PM).
6.  **Car Color Detector:** Traffic signal analyzer that counts cars/people and identifies blue vehicles using HSV logic.
